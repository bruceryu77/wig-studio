# WIG Studio - 가발 가상 착용 앱

손님이 스마트폰 카메라로 얼굴을 촬영하고, 다양한 가발 스타일을 가상으로 착용해볼 수 있는 웹 앱입니다.

## 기능

- 📷 **얼굴 촬영** - 전면/후면 카메라 지원
- 🤖 **얼굴 자동 인식** - AI가 얼굴 위치를 감지하여 가발을 자동 배치
- 💇 **7가지 가발 스타일** - 숏 밥, 긴 생머리, 웨이브, 컬, 포니테일, 픽시컷, 앞머리
- 🎚️ **크기/위치 조절** - 슬라이더로 가발 크기와 위치 미세 조정
- 💾 **결과 저장** - 착용 결과 이미지를 PNG로 저장

## 실행 방법

### 1. 로컬 서버 실행 (필수)

카메라 API는 HTTPS 또는 localhost에서만 동작합니다.

```bash
# Python이 설치되어 있다면
cd WIG
python -m http.server 8080

# 또는 Node.js가 있다면
npx serve .
```

브라우저에서 `http://localhost:8080` 접속

### 2. 스마트폰에서 사용

PC와 같은 Wi-Fi에 연결된 상태에서:
- PC의 IP 주소 확인 (예: `192.168.0.10`)
- 스마트폰 브라우저에서 `http://192.168.0.10:8080` 접속

> 실제 매장에서 사용하려면 HTTPS 호스팅이 필요합니다 (Netlify, Vercel 등 무료 호스팅 가능).

## 가발 이미지 교체

`assets/wigs/` 폴더의 SVG 파일을 실제 가발 사진(PNG)으로 교체할 수 있습니다.
같은 파일명으로 저장하면 앱에 자동 반영됩니다.

권장 이미지 규격:
- PNG 형식, 투명 배경
- 정면에서 본 가발 모양
- 약 400×400px 이상

## 파일 구조

```
WIG/
├── index.html          # 메인 페이지
├── css/style.css       # 스타일
├── js/app.js           # 앱 로직 (카메라, 얼굴인식, 가발합성)
├── assets/wigs/        # 가발 이미지
└── README.md
```
